# 🎒 Guide Enseignant — PEAN pour les Profs 🧑‍🏫

> **T'as pas besoin d'être un génie de l'ordi. T'as juste besoin de savoir cliquer et lire.**

---

## 🌟 C'est quoi PEAN ?

PEAN, c'est une **boîte à outils magique** pour faire passer des examens sans t'arracher les cheveux.

- Tu crées un examen 🧪
- Tu ajoutes des questions ✏️
- Les étudiants répondent 📱
- **L'ordinateur corrige tout tout seul** 🤖✨
- Toi, tu peux juste vérifier et mettre une jolie note 📊

C'est comme avoir un **petit robot assistant** qui fait tout le travail pénible pendant que tu bois ton café. ☕

---

## 🪪 Étape 1 : Créer ton compte

Avant de jouer, faut t'inscrire. C'est comme créer un personnage dans un jeu vidéo.

### Ce qu'il te faut :
Un **code d'invitation** (c'est un mot de passe spécial que ton admin te donne). Sans ça, tu peux pas entrer. C'est comme le code secret de la clubhouse. 🤫

### Les étapes :

1. Va sur le site PEAN 🌐
2. Clique sur **"Je suis un enseignant"** 👨‍🏫
3. Clique sur **"Créer un compte"** ➕
4. Remplis le petit formulaire :
   - ✨ **Code d'invitation** — Les 12 lettres magiques que ton chef t'a données
   - 👤 **Ton nom** — Ex: "Monsieur Dupont"
   - 📧 **Ton email** — L'adresse où tu reçois tes mails (vérifie bien !)
   - 🔑 **Mot de passe** — 8 lettres minimum. Pas "123456" hein ! On se concentre.
   - 🏫 **Ton école/université** — Le nom de là où tu travailles
   - 📚 **Ta matière** — Maths, Français, Science...
5. Clique sur **"S'inscrire"** ✅
6. Va voir tes **mails** (et vérifie les spams, le courrier se cache parfois) 📨
7. Clique sur le lien dans le mail — Bim, t'es inscrit ! 🎉

> ⚠️ **Si t'as pas de code d'invitation**, demande à ton directeur ou à l'admin. Sans code, tu restes devant la porte comme un livreur sans code d'immeuble. 🚪

---

## 🔓 Étape 2 : Se connecter

1. Tape ton **email** et ton **mot de passe**
2. Clique sur **"Connexion"**
3. 🎉 **T'es dans la place !**

### Si t'as la double authentification (2FA) 🔐

C'est un **super verrou supplémentaire**. Après ton mot de passe, on te demande un code secret à 6 chiffres qui change toutes les 30 secondes.

- Tu utilises **Google Authenticator** ou **Authy** sur ton téléphone
- L'appli affiche un code qui change tout le temps comme un feu clignotant
- Tu tapes ce code sur l'ordi
- Et voilà, t'es super protégé 🛡️

> 🔒 **Attention :** Si tu te trompes de mot de passe **5 fois**, ton compte se verrouille pendant **15 minutes**. Comme un coffre-fort qui dit "T'as trop essayé, repose-toi un peu". Va boire de l'eau. 💧

## 📊 Étape 3 : Ton Tableau de Bord (le QG)

Quand tu arrives sur PEAN, tu vois **ton tableau de bord**. C'est comme le **hall d'entrée de ta maison numérique**.

Tu vois tout en un coup d'œil :
- 📝 **Combien d'examens** tu as créés
- 👨‍🎓 **Combien d'étudiants** ont passé tes examens
- ⏳ **Combien de copies** tu dois encore vérifier
- 🕐 **Tes derniers examens** (clique dessus pour voir les détails)

C'est comme le rétroviseur de ta voiture : ça te montre d'un coup ce qui s'est passé. 👀

---

## 📝 Étape 4 : Créer un Examen (une "Session")

C'est le **plat principal** ! Tu veux faire passer un contrôle, un devoir, ou un examen ? On crée une **session**.

### Les étapes :

1. Va dans **"Mes sessions"** 🗂️
2. Clique sur **"Nouvelle session"** 🆕
3. Remplis les infos :

| Champ | C'est quoi ? | Exemple |
|-------|-------------|---------|
| **Titre** | Le nom de ton examen | "Contrôle sur les fractions" |
| **Type** | Quelle genre d'épreuve | Examen / Devoir / Rattrapage / Démo |
| **Durée** | Combien de temps en minutes | 60 (pour 1 heure) |
| **Barème** | Sur combien de points | 20 (comme à l'école) |
| **Matière** | Quelle matière | Mathématiques |
| **Date de début** | Quand ça commence (optionnel) | Demain à 8h |

4. Clique sur **"Créer"** ✅

🎉 **Bravo !** Ton examen est créé. Il est en mode **"brouillon"** (comme un dessin pas fini). Tu peux encore le modifier.

---

## ✏️ Étape 5 : Ajouter des Questions (les "Exercices")

Ton examen tout vide, c'est triste. Il faut lui **mettre des questions dedans**. Trois façons de faire :

### 🅰️ Option 1 : L'IA génère les questions (la méthode magique ✨)

C'est la façon la plus **cool**. Tu donnes ton cours à l'ordinateur, et il fabrique les questions tout seul !

1. Clique sur **"Générer avec l'IA"** 🤖
2. Donne-lui ton cours :
   - **Tu tapes ou tu colles** le texte de ton cours 📝
   - **Ou tu télécharges un fichier** (PDF, Word) 📄
3. Dis-lui ce que tu veux :
   - **Combien de questions** (3, 5, 10...)
   - **Quel type** : QCM ❓ / Question ouverte 📝 / Code 💻 / Un peu de tout
   - **Sur combien de points**
4. Clique sur **"Générer"** ✨
5. L'ordinateur réfléchit (il fait "Mmmmh...") et te sort des questions 🪄
6. **Tu vérifies** et tu ajustes si besoin (parce que toi, t'es le chef !)

> 💡 **Le secret magique :** Chaque question peut avoir **jusqu'à 6 versions différentes** (variantes). Les mots changent, les chiffres changent, les réponses sont mélangées. Comme ça, **chaque étudiant reçoit une copie différente**. Personne ne peut copier sur son voisin ! 🚫👀

### 🅱️ Option 2 : Tu crées les questions toi-même (à l'ancienne 👨‍🏫)

1. Clique sur **"Ajouter un exercice"** ➕
2. Remplis :
   - Le **titre** de la question
   - Les **instructions** (explique bien ce qu'il faut faire)
   - Le **type** : Question ouverte / QCM / Chiffres / Code
   - La **difficulté** : Facile 😊 / Moyen 😐 / Difficile 😰
   - Les **points**
   - La **réponse correcte** (pour que l'ordi sache ce qui est juste)
3. Si c'est un QCM, ajoute les choix A/B/C/D et dis quelle est la bonne réponse
4. Si tu veux, ajoute des **variantes** (des versions différentes)

### 🅲 Option 3 : Tu importes une vieille épreuve (recyclage ♻️)

T'as un vieux sujet d'examen en PDF ou Word ? Importe-le dans la machine !

1. Clique sur **"Téléverser une épreuve"** 📤
2. Choisis ton fichier
3. L'IA le lit et fabrique les questions automatiquement

**C'est magique !** 🪄

## 👨‍🎓 Étape 6 : Ajouter les Étudiants

Avant de lancer l'examen, faut dire **qui va passer l'examen**. Y'a 3 façons :

### Option 1 : Une liste toute prête 📋

1. Va dans **"Listes d'étudiants"**
2. Clique sur **"Nouvelle liste"** ➕
3. Choisis comment ajouter les noms :
   - **Fichier Excel ou CSV** — Télécharge un fichier avec les noms et matricules 📁
   - **Fichier PDF** — L'ordi lit le PDF et trouve les noms tout seul 🤖
   - **Tape à la main** — Ajoute chaque étudiant un par un ⌨️
4. Vérifie que tout est bon, puis valide ✅
5. Va dans ta session et associe la liste

### Option 2 : Tu tapes les noms directement pendant la création ✍️

Au moment de créer ta session, tu peux juste taper les noms et matricules dans l'écran.

### Option 3 : Une classe déjà prête 🏫

Si ton école a déjà mis en place des classes (avec des groupes d'étudiants), tu peux juste choisir la classe. Simple comme bonjour !

---

## 🚀 Étape 7 : Lancer l'Examen (C'est parti !)

Vérifie que :
- ✅ Les questions sont prêtes
- ✅ Les étudiants sont dans la liste
- ✅ Tout a l'air cool

Clique sur **"Lancer la session"** 🎯

Le statut devient **"active"** (comme un feu vert 🟢).

Les étudiants peuvent maintenant rejoindre avec un **code à 8 lettres** que l'écran leur montre.

> 🔐 **Code secret en plus :** Tu peux aussi créer un **code PIN à 6 chiffres** pour chaque étudiant. Comme ça, en plus du code de la session, l'étudiant doit donner son code personnel. La double sécurité, comme un coffre avec deux serrures ! 🔒🔒

---

## 👀 Étape 8 : Pendant l'Examen (Tu surveilles)

Quand l'examen tourne, tu peux voir **en direct** ce qui se passe :

- 🔵 **Bouton bleu** — L'étudiant n'a pas encore commencé (il attend)
- 🟡 **Jaune** — L'étudiant est en train d'écrire (il bosse)
- 🟢 **Vert** — L'étudiant a rendu sa copie (fini !)
- 🔴 **Rouge** — Problème ! L'étudiant a essayé de tricher ou de changer de fenêtre 🚨

Les soumissions arrivent **en temps réel**, comme des messages WhatsApp. 👨‍💻

> ⏰ **Pour finir l'examen :** Tu cliques sur **"Terminer la session"**. Les étudiants qui n'ont pas fini sont automatiquement soumis. Cache-cache terminé ! 🙈

## 🤖 Étape 9 : La Correction (Le robot fait le travail !)

C'est là que PEAN est vraiment **magique** ! 🪄 L'ordinateur corrige les copies à ta place.

### Correction automatique :

1. Va dans **"Corrections"** 🔍
2. Clique sur **"Corriger tout avec l'IA"** 🤖
3. L'IA lit chaque copie et donne :
   - Une **note** 📊
   - Des **commentaires** pour chaque question 💬
   - Un **pourcentage de confiance** (si le robot est sûr ou pas)
4. Le statut passe à **"Corrigé par IA"** ✅

### Vérification par toi (t'es le chef !) :

1. Ouvre une copie corrigée
2. Regarde ce que le robot a mis comme note
3. Si t'es pas d'accord, **tu changes** (t'es le patron !)
4. Tu peux ajouter des **annotations** (des petites notes sur la copie) :
   - 💬 Un commentaire
   - ✅ Une correction
   - 🖍️ Un surlignage
   - ⭐ Une félicitation ("Bravo !")
5. Tu valides en cliquant sur **"Publier"** 📢

### La grille d'évaluation (optionnel) :

Tu peux créer une **grille** avec des critères :
- "Raisonnement" → 5 points max
- "Résultat" → 3 points max
- "Présentation" → 2 points max

Pendant la correction, tu coches case par case. C'est comme une checklist ! ✅

### Correction rapide des QCM :

Si t'as que des QCM, tu peux cliquer sur **"Corriger les QCM"** — c'est encore plus rapide ! ⚡

## 📤 Étape 10 : Voir les Résultats et les Exporter

1. Va dans **"Résultats"**
2. Tu vois un **grand tableau** avec :
   - Nom de l'étudiant
   - Matricule
   - Note (celle du robot, la tienne, la finale)
   - Statut
3. Tu peux **exporter** tout ça en :
   - 📊 **Excel** (pour faire de jolis tableaux)
   - 📄 **PDF** (pour imprimer et afficher au tableau)
   - 📑 **CSV** (pour les pros de l'informatique)

## 📚 Étape 11 : La Bibliothèque (Tes documents)

C'est comme **ta bibliothèque personnelle** où tu ranges tous tes cours.

1. Va dans **"Bibliothèque"**
2. Clique sur **"Nouveau document"** ➕
3. Télécharge un fichier (PDF, Word, PowerPoint...)
4. L'ordinateur classe tout ça tout seul :
   - 📖 **Type** : Cours / TD / TP / Examen / Corrigé
   - 🏷️ **Matière** détectée automatiquement
   - 📊 **Niveau** suggéré
   - 🔑 **Mots-clés** importants
   - 📝 **Résumé** automatique
5. Tu peux ajuster si le robot s'est trompé

### Chercher dans tes documents :

- Tape un mot dans la **barre de recherche** 🔍
- Filtre par type, matière, niveau
- L'ordi cherche même dans le texte des documents (comme Google dans ta bibliothèque !)

---

## 💻 Étape 12 : Les Exercices de Code

Tu veux que les étudiants **programment** ? C'est possible !

1. Crée un exercice de type **"Code"** 💻
2. Choisis le langage : Python, JavaScript, Java, C++, etc.
3. Écris l'énoncé (ce que l'étudiant doit faire)
4. Ajoute des **tests** (des entrées qui donnent des sorties précises)
5. Tu peux donner une **solution de référence** (le corrigé parfait)

L'étudiant écrit son code, peut le tester, et quand il soumet, les tests vérifient si ça marche.

**Comme un juge automatique !** 🧑‍⚖️
