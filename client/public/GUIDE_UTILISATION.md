## 🤔 Tu sais pas qui tu es ?

- **T'es prof ?** → Va voir le [Guide Enseignant](GUIDE_ENSEIGNANT.md) 🧑‍🏫
- **T'es responsable de département ?** → Va voir le [Guide CD](GUIDE_CD.md) 🧭
- **T'as genre "tous les droits" mais t'es pas le big boss ?** → Va voir le [Guide Admin](GUIDE_ADMIN.md) 👑
- **T'as VRAIMENT tous les droits, le nom de ton rôle c'est "super_admin", tu peux même créer des universités ?** → Va voir le [Guide Super Admin](GUIDE_SUPER_ADMIN.md) ⚡
- **T'es étudiant ?** → Tu t'inscris pas, tu reçois juste un code de ton prof, tu tapes le code sur le site, et tu passes ton examen. **C'est tout.** Pas besoin de guide. 😎

## 🗺️ L'Arbre Magique des Rôles

```
👑 SUPER ADMIN ⚡ (il peut TOUT faire)
  └─ 👑 ADMIN (il gère toute l'école, sauf institutions et années)
      └─ 🧭 CD / CHEF DE DÉPARTEMENT (il gère les listes étudiants et surveille)
          └─ 🧑‍🏫 ENSEIGNANT (il fait les examens et les corrections)
              └─ 👨‍🎓 ÉTUDIANT (il passe juste l'examen — pas besoin de compte !)
```

Chaque rôle peut faire tout ce que les rôles en-dessous savent faire.
Un Super Admin peut aussi être enseignant, CD et Admin. **Mais pas l'inverse !** 🔄

## ❓ T'as besoin d'aide en urgence ?

| Situation | Solution |
|-----------|----------|
| 😵 Je comprends rien à mon guide | Relis-le, c'est fait exprès pour les nuls |
| 🔑 J'ai pas de code d'invitation | Va voir celui qui t'a donné ton compte |
| 💻 Ça marche pas | Appelle ton admin |
| 📱 J'ai perdu mon téléphone (et ma 2FA) | Admin ! Vite ! |
| 🤖 L'IA fait n'importe quoi | Corrige toi-même, t'es le chef |
| 🤷 J'ai changé d'avis sur un rôle | Pas de panique, retourne en mode Enseignant |

PEAN, c'est comme une **boîte à outils** : chaque outil a sa fonction. Utilise celui qui correspond à ton rôle, et tout ira bien. 🛠️✨
