# 🏰 Guide Administrateur — PEAN, le Jeu des Trônes Académique 👑

> **T'es le chef de toute la plateforme (ou presque). Les enseignants comptent sur toi. Les CD te respectent. Les Super Admins te regardent de loin. Ce guide est ta boussole.**

---

## 🌟 C'est qui l'Admin ?

Toi, t'es l'**administrateur**. T'es un ancien enseignant à qui on a donné des **super pouvoirs**. 🦸‍♂️

Tu peux faire **TOUT ce que fait un enseignant** + **TOUT ce que fait un CD** + **plein de trucs en plus** :

- ✅ Gérer les enseignants (les promouvoir, les rétrograder)
- 🏛️ Créer des filières et des matières
- 🏫 Organiser les classes
- 🔑 Créer des codes d'invitation (sans ça, personne peut s'inscrire)
- 📖 Voir le journal d'audit
- 📋 Gérer les listes d'étudiants
- 👀 Surveiller les incidents

Mais tu peux **PAS** :
- ❌ Créer des institutions (c'est pour le Super Admin)
- ❌ Créer des années académiques (c'est pour le Super Admin)

T'es comme le **directeur d'une école** : tu gères tout dans l'école, mais c'est le "ministère" (Super Admin) qui décide des grandes règles. 🏫

---

## 🚀 Activer le Mode Admin

1. Connecte-toi avec ton compte normal 👨‍💻
2. Regarde **en bas à gauche** — y'a un bouton qui dit "Mode Enseignant"
3. Clique dessus et choisis **"Mode Administrateur"** 🔄
4. **POW !** 💥 Le menu change : t'as des tonnes d'options en plus !

Pour revenir en enseignant, refais la même manip. C'est comme avoir une **double casquette** ! 🧢🧢

## 🏠 Le Tableau de Bord Admin

Quand t'arrives en mode Admin, tu vois un **grand tableau de bord** avec des chiffres :

- 👨‍🏫 **Combien d'enseignants** sont inscrits
- 📝 **Combien d'examens** ont été créés
- 👨‍🎓 **Combien d'étudiants** ont été évalués
- ✅ **Combien de corrections** ont été faites

C'est comme le **compteur de ta caisse de supermarché** : tout est compté ! 🧮

## 👨‍🏫 Gérer les Enseignants

### Voir la liste des profs

Tu as un grand tableau avec tous les enseignants :
- 📛 Leur nom
- 📧 Leur email
- 🏫 Leur école
- 🎭 Leur rôle (enseignant, admin, etc.)
- 📅 Quand ils se sont inscrits

Tu peux filtrer par école, par rôle, ou par statut.

### Promouvoir un enseignant en Admin

Tu trouves un collègue super compétent ? Tu veux lui donner des pouvoirs ?

1. Ouvre le profil de l'enseignant 👤
2. Clique sur **"Promouvoir administrateur"** ⬆️
3. Confirme — **Et bim !** Il devient admin ! 🎉

> ⚠️ **Attention :** Quand tu fais quelqu'un admin, cette personne peut tout gérer comme toi. Choisis bien !

### Rétrograder un Admin en enseignant

Quelqu'un a fait des bêtises ou a changé de rôle ?

1. Ouvre le profil de l'admin
2. Clique sur **"Rétrograder en enseignant"** ⬇️
3. Confirme — Il redevient un simple prof comme avant

## 🏛️ Organiser l'École (Hiérarchie Pédagogique)

Imagine une **poupée russe** : une grosse qui en contient une plus petite, qui en contient une encore plus petite... 🪆

Voilà comment PEAN organise les écoles :

```
🏛️ Institution (Université)
  └─ 📂 Filière (Informatique, Médecine, Droit...)
      └─ 📅 Année académique (2025-2026)
          └─ 📊 Niveau (L1, L2, L3, M1, M2...)
              └─ 🏫 Classe (Groupe A, Groupe B...)
                  └─ 👨‍🎓 Étudiants
```

### 🏛️ Les Filières

Une filière, c'est une **branche d'étude** : Informatique, Médecine, Droit, Commerce...

**Créer une filière :**
1. Va dans **"Filières"**
2. Clique sur **"Nouvelle filière"** ➕
3. Donne-lui un nom : "Informatique", "Génie Civil"...
4. Rattaché-la à une institution
5. Sauvegarde ✅

Tu peux **modifier** le nom, ou **supprimer** (si elle est vide).

### 📚 Les Matières

Les matières, c'est ce que les enseignants enseignent : Maths, Français, Physique...

**Créer une matière :**
1. Va dans **"Matières"**
2. Clique sur **"Nouvelle matière"** ➕
3. Donne un nom unique : "Mathématiques L1", "Algorithmique"...
4. Sauvegarde ✅

> 💡 Si tu supprimes une matière utilisée dans des examens, ça peut poser problème. Vérifie avant de supprimer !

### 📊 Les Niveaux d'Étude

Les niveaux, c'est : Licence 1, Master 2, Doctorat, etc.

**Créer un niveau :**
1. Va dans **"Niveaux d'étude"**
2. Clique sur **"Nouveau niveau"** ➕
3. Donne un nom : "Licence 1", "Master 2"...
4. Sauvegarde ✅

### 🏫 Les Classes

Une classe, c'est un **groupe d'étudiants** qui suivent les mêmes cours : "Groupe A", "Groupe B".

**Créer une classe :**
1. Va dans **"Spécialités"** (oui, le bouton s'appelle Spécialités mais c'est pour les classes) 🏷️
2. Clique sur **"Nouvelle classe"** ➕
3. Remplis :
   - **Nom** : "Groupe A", "TD1"...
   - **Filière** : Dans quelle filière ?
   - **Année académique** : Quelle année ?
   - **Niveau** : Quel niveau ?
4. Sauvegarde ✅

**Ajouter des étudiants dans une classe :**
1. Ouvre la classe
2. Tu peux :
   - ✍️ Ajouter manuellement (Nom + Matricule)
   - 📁 Importer un fichier CSV
3. Sauvegarde ✅

Les enseignants pourront **utiliser ces classes** quand ils créent leurs sessions. Magique ! 🪄

## 🔑 Les Codes d'Invitation

C'est **très important** ! Sans code d'invitation, PERSONNE ne peut s'inscrire. C'est la **porte d'entrée** de PEAN. 🚪

### Créer des codes

1. Va dans **"Codes d'invitation"**
2. Clique sur **"Générer des codes"** 🎰
3. Dis combien tu veux de codes (max 100 à la fois)
4. Choisis une **date d'expiration** (ils seront périmés après cette date) ⏰
5. Clique sur **"Générer"** — Et hop, des codes apparaissent ! ✨

### Gérer les codes

Tu vois un tableau avec :
- 🔑 Le code (12 caractères mystérieux)
- 📅 La date de création
- ✅ Le statut : Actif / Utilisé / Révoqué / Expiré
- 👤 Qui a utilisé le code (si c'est le cas)

**Actions possibles :**
- 📋 **Copier un code** — Pour le donner à un prof
- 📋 **Copier tout le lot** — Si t'as 50 codes à distribuer
- 🚫 **Révoquer un code** — Le rendre inutilisable (si tu changes d'avis)
- 🗑️ **Supprimer un code** — Le faire disparaître

### Les chiffres utiles

Tu vois aussi des **statistiques** :
- 📊 Combien de codes ont été créés au total
- ✅ Combien ont été utilisés
- 🆓 Combien sont encore disponibles
- 📈 Le taux d'utilisation (tu gères bien ?)

## 📖 Le Journal d'Audit (Le Grand Livre)

Tout ce qui se passe sur PEAN est écrit dans un **journal invisible**. Toi, tu peux le lire. 📕

1. Va dans **"Journal d'audit"**
2. Tu vois danser devant tes yeux :
   - 🕐 **Quand** l'action a eu lieu
   - 👤 **Qui** l'a faite (email de la personne)
   - 🔧 **Quoi** : connexion, création, modification, suppression...
   - 🎯 **Sur quoi** : session, utilisateur, code...
   - 🌐 **Adresse IP** (l'adresse de l'ordinateur utilisé)

**Tu peux filtrer :**
- Par type d'action 🔍
- Par personne 👤
- Par ressource 📦
- Par période 📅 (entre deux dates)

> 💡 **Utilité :** Si quelqu'un fait n'importe quoi, tu peux retrouver exactement ce qui s'est passé. Comme une caméra de surveillance, mais pour les clics. 📹

---

## 🚨 Les Incidents de Sécurité

Quand un étudiant fait une bêtise pendant un examen (changer de fenêtre, essayer de tricher...), c'est noté comme **incident**.

1. Va dans **"Incidents"**
2. Tu vois tous les incidents :
   - 🏷️ **Type** : changement d'onglet, sortie d'examen...
   - 👤 **Étudiant** concerné
   - 📝 **Session** concernée
   - ⚠️ **Sévérité** (grave ou pas grave)

Tu peux voir qui triche ou qui a des problèmes techniques. 📊
