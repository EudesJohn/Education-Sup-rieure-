## 🏛️ Pouvoir EXCLUSIF n°1 : Gérer les Institutions (Établissements)

### C'est quoi une institution ?

Une **institution**, c'est le nom de ton école, université, ou établissement. Exemples :
- 🏛️ Université de Paris
- 🏛️ Institut de Génie Logiciel
- 🏛️ École Supérieure de Commerce

**C'est le point de départ de TOUT.** Avant de créer des filières, des classes ou d'ajouter des enseignants, il faut que l'institution existe. C'est la **fondation** sur laquelle tout est construit.

### Créer une institution

1. Va dans **"Établissements"** 🏛️ (dans le menu Super Admin)
2. Clique sur **"Nouvel établissement"** ➕
3. Tape le **nom** de l'établissement
4. Tu peux ajouter une **description** (optionnelle) : "Université publique fondée en 1960"
5. Clique sur **"Créer"** ✅

**Bravo !** Tu viens de créer une institution. Les admins pourront maintenant ajouter des filières dedans. 👏

### Modifier une institution

1. Trouve l'institution dans la liste
2. Clique sur le crayon ✏️ (modifier)
3. Change le nom ou la description
4. Sauvegarde ✅

### Supprimer une institution

1. Trouve l'institution dans la liste
2. Clique sur la poubelle 🗑️
3. **⚠️ ATTENTION :** Une confirmation s'affiche
4. Si tu confirmes, **tout disparaît** : l'institution, ses filières, ses niveaux, ses classes, les données associées...
5. Ne clique "Oui" que si t'es **vraiment sûr**

> 🚨 **Règle numéro 1 du Super Admin :** Ne supprime JAMAIS une institution qui a encore des enseignants ou des étudiants dedans. Vérifie avant. **Vérifie 2 fois, supprime 1 fois.**

### Voir la liste des institutions

Quand tu arrives dans "Établissements", tu vois toutes les institutions avec :
- 🏛️ **Leur nom**
- 📝 **Leur description** (si tu en as mis une)
- 👨‍🏫 **Nombre d'enseignants** rattachés
- 🎓 **Nombre d'étudiants** dans les classes
- 📅 **Date de création**

## 📅 Pouvoir EXCLUSIF n°2 : Gérer les Années Académiques

### C'est quoi une année académique ?

C'est l'année scolaire ou universitaire : "2025-2026", "2026-2027"... C'est le **calendrier** de toute la plateforme.

### Créer une année académique

1. Va dans **"Années académiques"** 📅 (dans le menu Super Admin)
2. Clique sur **"Nouvelle année"** ➕
3. Donne-lui un **nom** : "2025-2026", "2026-2027"...
4. Tu peux ajouter une **date de début** (optionnel) : "01/09/2025"
5. Tu peux ajouter une **date de fin** (optionnel) : "31/08/2026"
6. Coche **"Année en cours"** si c'est l'année actuelle
7. Clique sur **"Créer"** ✅

### Marquer l'année en cours

Il y a toujours une année académique qui est "l'année en cours". Tout le monde l'utilise par défaut.

1. Trouve l'année dans la liste
2. Clique sur **"Marquer comme année en cours"** ⭐
3. 👑 **Voilà !** C'est l'année officielle

### Modifier une année académique

1. Clique sur le crayon ✏️
2. Change le nom, les dates
3. Sauvegarde ✅

### Supprimer une année académique

1. Vérifie qu'elle n'est pas utilisée dans des sessions
2. Clique sur la poubelle 🗑️
3. Confirme
4. **Disparue !**

> 💡 **Astuce :** Crée toujours l'année **suivante** avant la fin de l'année en cours. Comme ça, la transition est toute douce.

## 🗺️ L'Arche de la Création (par où commencer ?)

Quand tu mets en place PEAN dans un nouvel établissement, **l'ordre est super important**. C'est comme construire une maison : tu commences par les fondations, puis les murs, puis le toit.

```
ÉTAPE 1 : 🏛️ INSTITUTION
   ↓
ÉTAPE 2 : 📅 ANNÉE ACADÉMIQUE
   ↓
ÉTAPE 3 : 👑 Les ADMINS créent les Filières, Niveaux, Classes
   ↓
ÉTAPE 4 : 🔑 Les ADMINS génèrent les Codes d'Invitation
   ↓
ÉTAPE 5 : 🧑‍🏫 Les ENSEIGNANTS s'inscrivent avec les codes
   ↓
ÉTAPE 6 : 👨‍🎓 Les ÉTUDIANTS passent leurs examens
```

**Sans institution → rien n'existe.**
**Sans année académique → les classes peuvent pas être créées.**
**Sans filières → les matières ont pas de maison.**
**Sans codes d'invitation → personne peut s'inscrire.**

T'as compris le principe ? **Tout part de toi.** 🏆

## 💡 Les 10 Commandements du Super Admin

### 1️⃣ Fais confiance, mais vérifie ✅
T'as le pouvoir de tout faire, mais tu dois pas tout faire. Nomme des **admins de confiance** dans chaque institution.

### 2️⃣ Délègue aux admins 👑
Les admins peuvent gérer les filières, matières, classes et codes. **Laisse-les faire leur travail.** Toi, tu t'occupes des institutions et des années.

### 3️⃣ Gère bien les codes d'invitation 🔑
Les codes d'invitation, c'est la **porte d'entrée** de PEAN. Trop de codes = n'importe qui peut s'inscrire. Pas assez de codes = les profs peuvent pas travailler.

### 4️⃣ Crée les années à l'avance 📅
En novembre 2025, crée déjà "2026-2027". Comme ça, à la rentrée, tout est prêt. Personne ne te court après.

### 5️⃣ Check le journal d'audit régulièrement 👀
Une fois par semaine, jette un œil. Pas pour espionner, mais pour **voir si tout va bien**. C'est comme vérifier les caméras de sécurité de ton immeuble.

### 6️⃣ Nomme des admins par institution 🏛️
Si t'as 3 universités, nomme au moins **un admin par université**. Tu peux pas gérer 3 universités tout seul.

### 7️⃣ Supprime avec précaution 🗑️
Avant de supprimer une institution, vérifie qu'elle est vide. Une fois supprimée, **c'est fini**. Y'a pas de "Ctrl+Z" qui remet tout.

### 8️⃣ Communique avec tes équipes 📞
Préviens les admins quand tu crées une nouvelle institution ou année. Ils sauront quoi faire ensuite.

### 9️⃣ Teste avant de lancer 🧪
T'as créé une institution ? Demande à un admin de tester si tout marche. Vérifie qu'il peut créer une filière dedans.

### 🔟 Amuse-toi un peu 🎉
T'as le plus haut niveau d'accès. Mais n'oublie pas : PEAN est fait pour aider les étudiants à apprendre. C'est ça, le vrai objectif.

## ⚠️ Les Pièges à Éviter ABSOLUMENT

### 🚨 Piège n°1 : Supprimer une institution utilisée
Une institution avec 3 filières, 50 enseignants et 2000 étudiants ?
**NE LA SUPPRIME PAS** sans avoir tout vidé avant.

### 🚨 Piège n°2 : Oublier de créer l'année académique
Si t'as pas créé l'année "2025-2026", personne peut créer de classes. Les admins vont t'appeler. **Tous les jours.**

### 🚨 Piège n°3 : Donner des codes à n'importe qui
Un code d'invitation = l'accès à PEAN pour un enseignant. Si tu donnes 50 codes à une personne non autorisée, **50 inconnus peuvent s'inscrire**.

### 🚨 Piège n°4 : Faire tout tout seul
T'es Super Admin, pas Super-Esclave. Nomme des admins pour t'aider.

### 🚨 Piège n°5 : Changer un paramètre sans prévenir
Si tu modifies une institution ou une année, préviens les gens concernés. La communication, c'est la clé. 🗝️
